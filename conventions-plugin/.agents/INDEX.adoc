== Contexte Global — Pourquoi Conventions ?

La code review globale des 26 boroughs a revele **6 familles de redondances** dans les fichiers `build.gradle.kts` :

. **Test configuration** : `useJUnitPlatform()`, `testLogging`, `jvmArgs`, `java { withSourcesJar(); withJavadocJar() }` — identique dans ~20 projets
. **FunctionalTest source set** : ~25 lignes identiques dans ~12 projets (sourceSet + dependencies + task + check.dependsOn)
. **Cucumber BDD setup** : `features/scenarios` source dirs + task `cucumberTest` + `excludeEngines` + filter — identique dans ~8 projets
. **Maven Central POM** : `developers { id = "cccp-education" }` + `scm {}` + `licenses {}` — identique dans ~15 projets
. **Signing conditionnel** : `signing { if (CI != "true" && !SNAPSHOT) sign(pubs) }` — identique dans ~15 projets
. **buildscript resolutionStrategy annotations** hack — identique dans ~5 projets

Gain estime : **~1200 lignes de build supprimees**, **1 source de verite**, **versions harmonisees**.
